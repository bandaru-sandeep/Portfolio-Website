## ğŸ“– How to Run Locally

Since this is a static website, you don't need to install complex dependencies like Node.js or Python environments to view it.

### Option 1: Direct File Open
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/bandaru-sandeep/Portfolio-Website.git
    ```
2.  Navigate to the project folder.
3.  Double-click `index.html` to open it in your default web browser.

### Option 2: Using VS Code Live Server (Recommended)
for a better development experience:
1.  Open the project folder in **VS Code**.
2.  Install the **Live Server** extension (by Ritwick Dey).
3.  Right-click on `index.html` and select **"Open with Live Server"**.
4.  The site will launch at `http://127.0.0.1:5500/`.
