Stampede Prediction and Avoidance Using Deep Learning

UROP Project | SRM University-AP | Dec 2025

Introduction

Stampedes are among the worst failures of crowd management. This project proposes a multi-mode, real-time proactive stampede detection system designed to identify precursor conditions rather than just detecting the stampede itself. It addresses the gap between perception and timely action by monitoring crowd behavior across the entire scene.

Problem Definition

The central problem addressed is the early identification of precursor conditions. Dangerous dynamics often originate in low-density peripheral areas, not just dense cores. Localized panic and reverse flows can escalate rapidly. Manual monitoring struggles to detect these subtle distributed changes in real time.

Proposed Solution

The solution adopts a four-stage multi-modal pipeline:

  • Reference Image Library: 5 representative frames from historical stampedes serve as visual templates.
  • Fast Visual Pre-Screening: UAV captures live video at 10 FPS. Frames are processed using perceptual hashing and compared with reference images.
  • Emergency Formula with RFID Fusion: If visual similarity > 40%, the system retrieves RFID-based indicators (density, movement) and computes a risk score.
  • Entropy-Based Confirmation: The risk score is converted into an entropy measure. If outside safe range < 0.06 or> 0.94, an alarm is triggered.

Methodology

The system uses specific algorithms for efficiency:

  • Perceptual Hashing: Frames are converted to grayscale, downsampled to 8x8, and hashed (64-bit). Similarity is measured using Hamming distance.
  • Emergency Formula: stp = K * Dt * (Rm * Cm)² * Pr * St² * Hb * ms * HV. This integrates density (Dt), reverse movement (Rm), cross-movement (Cm), pressure (Pr), etc.
  • Entropy Verification: Entropy = -p * log(p), where p is normalized instability probability. This reduces false alarms.

Implementation & Results

The system was implemented as a multi-stage pipeline. The combination of lightweight visual analysis, multi-sensor data fusion, and rule-based confirmation provided a reliable approach to early warning. The frame skipping optimization reduced processing load by 20-30% in stable conditions. The system successfully triggers a hooter alarm when entropy crosses critical thresholds, enabling immediate response.

Conclusion

This project demonstrates a proactive technology-based remedy for crowd safety. By integrating simple visual screening with complex sensor fusion, it achieves high-speed detection capable of anticipating stampedes in real-time. It minimizes false alarms and provides actionable intelligence to security personnel.
